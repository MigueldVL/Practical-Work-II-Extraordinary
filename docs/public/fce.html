Fundamentals of Computer Engineering (FCE)

Fundamentals of Computer Engineering is a subject offered to double-degree students at UFV. It's worth 3 ECTS and spans one semester, combining theoretical knowledge with practical web development and computing foundations.

Theoretical Content

  • Computational Engineering in Society – Impact, ethics, and future trends
  • Software and Data Concepts – Basics of information, software, databases
  • Internet of Things – Introduction to interconnected devices and sensors
  • Blockchain & Data Protection – Overview of digital security and privacy

Practical Content

  • Terminal & Git – Command line and repository version control via GitHub
  • HTML Development – Building structured web pages using HTML5
  • CSS Integration – Styling and layout using external CSS files
  • Publishing on GitHub Pages – Sharing and hosting your work online

Additional Goals

  • Explore the evolution of technology and its ethical implications
  • Understand the human dimension behind engineering and innovation
  • Connect your career interests to real-world applications
  • Familiarize with technological challenges in Spain and the EU
